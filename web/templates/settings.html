Agent update URL
Where your agent builds are hosted (this fork). Accepts: a direct file URL (e.g. https://my.site/klever-agent-linux) if all your servers share one platform; a template with {os}/{arch} (e.g. https://my.site/klever-agent-{os}-{arch}); or a base URL ending in / (then it appends klever-agent-<os>-<arch>, .exe on windows). Used by "Download from custom source" in the Agent Update dialog.
